\subsection{Detailed Description}
X.\+509 decoder context. 

This structure is {\itshape not} for X.\+509 validation, but for extracting names and public keys from encoded certificates. Intended usage is to use (self-\/signed) certificates as trust anchors.

Contents are opaque and shall not be accessed directly.
